HOW TO RUN
Install Python 3
Run main.py

HOW TO PLAY / CONTROLS
Arrow keys to move
Space to place a bomb

LEVEL EDITOR
Feel free to change the PNGs in the 'sections' folder to edit the level generation.
You can even add more sections, just follow the naming pattern!
